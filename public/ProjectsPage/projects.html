  • Built a Compiler

    For my compilers class I built a top down recursive descent compiler. I had to translate the given grammar into an LL(1) grammar. Then, I implemented the phases of a compiler: lexing, parsing, static analysis, code generation, optimizing, and register allocation. I used a bottom up greedy algorithm for register allocation to spill registers into memory that I would be using furthest in the future.
